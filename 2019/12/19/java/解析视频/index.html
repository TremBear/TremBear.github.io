分析结构

播放视频，会发现视频文件是由一个个的ts文件拼接而来的。

其中，每一个小的ts是一个视频片段，只需要拼接就行了。
但是具体包含哪些片段呢，有另外一个请求可以看到。

在这个地方，可以看到由m3u8的请求可以看到，有四个版本的视频源，分别为高低清画质的。然后访问高画质的视频源，可以获取所有的ts文件。
最后，拼接一下ts文件即可。
